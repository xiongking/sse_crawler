从上海证券交易所下载股票的公告pdf文件

## 下载代码

```
git clone https://github.com/xiongking/sse_crawler.git
```

## 创建虚拟环境
```
python -m venv sse_crawler_env
```
## 进入虚拟环境
```
sse_crawler_env\Scripts\activate
```

## 安装依赖
```
pip install -r requirements.txt
```

## 开始运行

```
python sse_crawler.py
```

输入六位数股票代码根据菜单提示操作即可。
